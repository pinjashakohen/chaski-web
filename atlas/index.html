The Problem

Notes everywhere, findable nowhere

Ideas land in a dozen apps — chat threads, sticky notes, half-finished docs. By the time you need one, you can't remember where you put it, and search across all of them doesn't exist.

screenshot_0421.png untitled note that link somewhere buried in a thread no full-text search
What Atlas Does

A capture box that never loses a thing

Four things, done well. Everything below is live in the app today — no waitlist, no "coming soon."

01 · CAPTURE

Add a document

Give it a title and content — a note, a summary, a source. Saved in one action, no formatting required.

02 · SEARCH

Full-text, instantly

Type a query; Atlas filters your documents and returns the matches — newest first. Searchable the moment you save, with no indexing delay.

03 · TAG

Light-touch tags

Attach tags to a document to group related material. Simple structure, only when you want it.

04 · PRIVATE

Yours alone

Every document is isolated to your account by row-level security in Postgres. Nobody else's queries can reach your notes.

Honest status

Live, and deliberately small

Live now Atlas is shipped and running inside the Pinjas app at app.pinjasemet.com/home/atlas. Capture, full-text search, tags, and per-account isolation all work today. Search input was hardened on 2026-07-07. It is a focused capture-and-find tool — not a graph or a second-brain wiki, and it doesn't claim to be.
